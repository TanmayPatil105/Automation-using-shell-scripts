# Wifi CLI simplified

## For Linux OS only

```Last Updated on 13-09-2022```

``` Try wifi --help after installation```

### Installation

```
$ git clone https://github.com/TanmayPatil105/Automation-using-shell-scripts.git
```
```
$ cd Automation-using-shell-scripts
```
```
$ sudo chmod +x installer.sh
```
```
$ ./installer.sh
```
### Uninstallation

```
$ ./installer.sh remove
```

### Usage 

```wifi on```
```wifi off```
```wifi list```
```wifi connect '$SSID'```
```wifi pass```
```wifi saved```
